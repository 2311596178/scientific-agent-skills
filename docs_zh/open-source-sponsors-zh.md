# 支持我们所依赖的开源项目

Scientific Agent Skills 站在巨人的肩膀上。本仓库中的 138 个技能依托数十个卓越的开源项目，这些项目由全球各地敬业的开发者和研究社区创建并维护。

**如果您认为这些技能对您有价值，请考虑支持那些使它们成为可能的开源项目。**

---

## 如何支持开源

1. **在 GitHub 上为仓库点 Star** — 免费且有助于项目获得更多关注
2. **直接赞助维护者** — 通过 GitHub Sponsors、Open Collective 或项目特定的捐赠页面
3. **贡献代码、文档或错误报告**
4. **在您的出版物中引用这些项目**
5. **与同事分享这些项目**
