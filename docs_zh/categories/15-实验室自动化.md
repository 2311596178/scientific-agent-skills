# 分类十五：实验室自动化

> 本分类包含 **1 个技能**，提供硬件无关的纯 Python SDK 用于自动化实验室的液体处理和设备控制。

---

## 分类概述

实验室自动化是连接计算设计与实验执行的桥梁。PyLabRobot 作为本分类的唯一技能，提供了极其广泛的硬件支持，从液体处理机器人到分析设备，使其成为实验室数字化转型的核心工具。

---

## 技能清单

### 1. pylabrobot（实验室自动化硬件控制）

**一句话简介**：硬件无关的纯 Python SDK，用于自动化和自主实验室。

**详细描述**：
PyLabRobot 提供统一接口控制液体处理机器人（Hamilton STAR/STARlet、Opentrons OT-2、Tecan EVO）、读板器（BMG CLARIOstar）、振荡加热器、培养箱、离心机、泵和天平。关键特性包括：模块化的资源管理系统（板、吸头、容器，层次化台架布局和 JSON 序列化）、综合液体处理操作（吸取、分配、转移、连续稀释、板复制，自动吸头和体积追踪）、后端抽象（跨不同机器人的硬件无关协议）、ChatterboxBackend（无需硬件的协议模拟和测试）、基于浏览器的 3D 台架状态实时可视化、跨平台支持（Windows、macOS、Linux、Raspberry Pi）。

**典型使用场景**：
- 自动化样品制备
- 高通量筛选
- 连续稀释协议
- 读板工作流
- 实验室协议开发和验证
- 机器人液体处理自动化
- 状态跟踪和持久化的可复现实验室自动化
