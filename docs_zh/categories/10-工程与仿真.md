# 分类十：工程与仿真

> 本分类包含 **4 个技能**，覆盖数值计算、计算流体力学、离散事件仿真和符号数学。

---

## 分类概述

工程与仿真技能为科学研究和工程设计提供了强大的计算工具。从 MATLAB/Octave 的矩阵运算到 FluidSim 的伪谱法流体力学模拟，从 SimPy 的离散事件系统建模到 SymPy 的精确符号计算，本分类覆盖了从数值到符号、从连续到离散的多种仿真范式。

---

## 技能清单

### 1. matlab（数值计算环境）

**一句话简介**：用于矩阵运算、数据分析、可视化和科学计算的数值计算环境。

**详细描述**：
MATLAB 是商业软件，GNU Octave 是免费开源替代方案。提供矩阵操作（创建、操作、线性代数）、综合数学（特征值、SVD、FFT、ODE、优化、统计）、2D/3D 可视化（plot、surf、contour）、数据导入/导出（CSV、Excel、MAT、图像）、编程结构（函数、脚本、控制流、OOP）、信号处理（FFT、滤波、卷积）和 Python 集成。支持向量化操作、匿名函数、表格和元胞数组。Octave 与大多数 MATLAB 脚本兼容。

**典型使用场景**：
- 数值模拟
- 信号处理
- 图像处理
- 控制系统
- 统计分析
- 算法原型

---

### 2. fluidsim（计算流体力学）

**一句话简介**：使用伪谱法和 FFT 的高性能计算流体力学（CFD）面向对象 Python 框架。

**详细描述**：
FluidSim 提供周期域方程求解器：2D/3D 不可压缩 Navier-Stokes 方程（有/无分层）、浅水方程和 Föppl-von Kármán 弹性板方程。特性包括 Pythran/Transonic 编译（性能接近 Fortran/C++）、MPI 并行化、分层参数配置、综合输出管理（HDF5 物理场、空间均值、能谱）、自定义强迫机制、灵活初始条件、在线和离线可视化，以及与 ParaView/VisIt 的 3D 可视化集成。

**典型使用场景**：
- 2D/3D 湍流研究
- 分层海洋和大气流动
- 地球物理流动（科里奥利效应）
- 涡动力学
- 直接数值模拟（DNS）
- 参数研究

---

### 3. simpy（离散事件仿真）

**一句话简介**：用于建模具有进程、队列和资源争用系统的基于进程的离散事件仿真框架。

**详细描述**：
SimPy 支持基于生成器的进程定义、多种资源类型（Resource、PriorityResource、PreemptiveResource、Container、Store）、事件驱动调度、进程交互机制（信号、中断、并行/顺序执行）、与壁钟时间同步的实时仿真，以及利用率、等待时间和队列统计的综合监控。

**典型使用场景**：
- 制造系统仿真
- 服务运营优化
- 网络流量建模
- 物流和供应链
- 排队系统分析

---

### 4. sympy（符号数学）

**一句话简介**：使用数学符号而非数值近似进行精确计算的 Python 符号数学库。

**详细描述**：
SymPy 提供符号代数（化简、展开、因式分解）、微积分（导数、积分、极限、级数）、方程求解（代数、微分、方程组）、矩阵和线性代数（特征值、分解）、物理（经典力学、量子力学、矢量分析）、数论、几何、组合数学、逻辑和集合、统计、特殊函数，以及代码生成（lambdify 到 NumPy/SciPy、C/Fortran 代码生成、LaTeX 输出）。

**典型使用场景**：
- 精确符号推导
- 代数方程求解
- 微分和积分符号计算
- 生成 LaTeX 公式
- 教学和研究中的符号数学
- 符号到数值的科学计算流程
