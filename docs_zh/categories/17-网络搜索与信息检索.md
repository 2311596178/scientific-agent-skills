# 分类十七：网络搜索与信息检索

> 本分类包含 **2 个技能**，提供高质量网络搜索和网页内容提取能力，特别针对科学和技术内容优化。

---

## 分类概述

在信息爆炸的时代，高效准确地获取网络信息是科学研究的重要能力。本分类的技能专注于科学和技术内容的搜索：Exa Search 提供针对学术论文和研究内容的高质量搜索，而 Parallel Web 则提供综合的网页搜索、内容提取和深度研究能力。两者互补，分别适用于精确学术检索和广泛网络调研。

---

## 技能清单

### 1. exa-search（Exa 科学网络搜索）

**一句话简介**：通过 Exa API 进行高质量网络搜索，特别针对科学和技术内容优化。

**详细描述**：
Exa Search 专为科学和技术内容设计，支持学术过滤（`category="research paper"` 和学术域名白名单）和批量 URL 提取。相比通用搜索引擎，Exa 在检索学术文献、技术文档和研究成果时提供更精确的结果。支持语义搜索，能够基于内容含义而非仅关键词进行匹配。

**典型使用场景**：
- 高质量学术文献检索
- 技术文档搜索
- 研究论文发现
- 科学内容验证
- 批量网页内容提取

---

### 2. parallel-web（并行网络搜索）

**一句话简介**：使用 Parallel Chat API 和 Extract API 搜索网页、提取 URL 内容并运行深度研究。

**详细描述**：
Parallel Web 提供综合的网络搜索、内容提取和深度研究能力。支持网页搜索、研究查询和带合成摘要与引用的一般信息收集。相比传统搜索，它能对多个来源的信息进行综合分析，生成结构化的研究摘要。

**典型使用场景**：
- 综合网络调研
- 多来源信息综合
- 事实核查
- 背景研究
- 趋势分析
- 竞争对手/竞品调研
