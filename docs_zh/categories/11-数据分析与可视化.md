# 分类十一：数据分析与可视化

> 本分类包含 **11 个技能**，覆盖大规模数据处理、地理空间分析、网络分析、科学可视化和高性能 DataFrame 操作。

---

## 分类概述

数据分析与可视化是科学研究的通用基础能力。本分类从大规模数据并行处理（Dask、Vaex、Polars）到地理空间分析（GeoPandas、GeoMaster），从网络科学（NetworkX）到发表级可视化（Matplotlib、Seaborn、Scientific Visualization），再到统计探索（Exploratory Data Analysis、Statistical Analysis），形成了一个完整的数据分析工具链。特别值得注意的是，本分类包含多个高性能 DataFrame 库（Polars、Dask、Vaex），分别针对不同规模和类型的数据处理需求。

---

## 技能清单

### 1. matplotlib（科学绘图）

**一句话简介**：用于创建发表级静态、动画和交互式可视化的综合 Python 绘图库。

**详细描述**：
Matplotlib 支持多种绘图类型（线图、散点图、条形图、直方图、等高线图、3D 图）、广泛的自定义选项（颜色、字体、样式、布局）、多种后端（PNG、PDF、SVG、交互式后端）、LaTeX 数学符号集成，以及与 NumPy 和 pandas 的集成。包括 pyplot（MATLAB 风格接口）、artist 层（精细控制）和 backend 层（渲染）。

**典型使用场景**：
- 科学图表创建
- 数据可视化
- 探索性数据分析
- 发表级图形

---

### 2. seaborn（统计可视化）

**一句话简介**：具有数据集导向接口、自动置信区间和发表级主题的数据集统计可视化。

**详细描述**：
Seaborn 建立在 Matplotlib 之上，提供高级统计可视化接口。特性包括自动置信区间、色盲安全调色板、多面板图形综合支持、分布比较、相关矩阵、回归图和探索性分析。与 pandas DataFrame 深度集成，一行代码即可生成复杂的统计图形。

**典型使用场景**：
- 统计分析和可视化
- 数据分布探索
- 相关性分析热图
- 回归分析可视化
- 多变量数据探索

---

### 3. scientific-visualization（科学可视化最佳实践）

**一句话简介**：使用 matplotlib 和 seaborn 创建发表级科学图形的最佳实践和模板。

**详细描述**：
该技能提供统计图形的最佳实践指导：自动置信区间、色盲安全调色板、多面板图形、热图和期刊特定格式。涵盖图形设计原则、颜色选择策略、统计可视化规范和学术出版要求。

**典型使用场景**：
- 准备期刊投稿图形
- 遵循领域特定的可视化规范
- 统计图形标准化
- 色盲友好的科学可视化

---

### 4. dask（并行计算）

**一句话简介**：用于大于内存数据集的并行计算，支持分布式 DataFrames、Arrays、Bags 和 Futures。

**详细描述**：
Dask 将 NumPy、Pandas 和原生 Python 代码扩展到多核和分布式环境。支持任务调度、延迟计算、内存管理和分布式执行。与现有 Python 生态系统无缝集成，可以在笔记本上开发和在生产集群上扩展。

**典型使用场景**：
- 大于内存的数据集处理
- 分布式机器学习
- 大规模数据并行分析
- 云原生数据处理

---

### 5. polars（高性能 DataFrame）

**一句话简介**：用 Rust 编写的高性能 DataFrame 库，具有 Python 绑定。

**详细描述**：
Polars 提供延迟评估查询优化、高效内存使用和并行处理。支持 DataFrame 操作（过滤、分组、连接、聚合）、大数据集（大于 RAM）、与 pandas 和 NumPy 的集成、表达式 API 进行复杂变换，以及多种数据格式（CSV、Parquet、JSON、Excel、Arrow）。通常比 pandas 快 5-30 倍。

**典型使用场景**：
- 大规模数据处理
- ETL 流程
- 高性能数据分析
- 内存受限环境

---

### 6. vaex（核外 DataFrame）

**一句话简介**：用于惰性核外 DataFrames 的高性能 Python 库，处理大于可用 RAM 的表格数据集。

**详细描述**：
Vaex 通过内存映射文件（HDF5、Apache Arrow）实现每秒处理超过 10 亿行。特性包括即时文件打开、跨数十亿行的高效聚合、无需采样的交互式可视化、ML 流程（缩放器、编码器、PCA），以及与 pandas/NumPy/Arrow 的无缝集成。包含 vaex.ml 框架，支持特征缩放、分类编码、降维，以及与 scikit-learn/XGBoost/LightGBM/CatBoost 的集成。

**典型使用场景**：
- 处理 GB 到 TB 级数据集
- 大规模数据的快速统计聚合
- 数十亿行数据集的可视化
- 大数据 ML 流程

---

### 7. geopandas（地理空间数据分析）

**一句话简介**：扩展 pandas 用于处理地理空间矢量数据的 Python 库。

**详细描述**：
GeoPandas 提供 GeoDataFrame 和 GeoSeries 数据结构。特性包括：空间文件格式读写（Shapefile、GeoJSON、GeoPackage、PostGIS、Parquet，Arrow 加速 2-4 倍 I/O）、几何操作（缓冲、简化、质心、凸包）、空间分析（空间连接、最近邻连接、叠加操作、 dissolve）、坐标参考系统（CRS）管理、静态和交互式地图可视化（matplotlib、folium）。

**典型使用场景**：
- 空间数据操作
- 缓冲区分析
- 空间连接
- 等值区域图
- PostGIS 数据库集成

---

### 8. geomaster（综合地理空间科学）

**一句话简介**：涵盖遥感、GIS、空间分析和地球观测机器学习的综合地理空间科学技能。

**详细描述**：
GeoMaster 支持卫星影像处理（Sentinel、Landsat、MODIS、SAR、高光谱）、矢量和栅格数据操作、空间统计、点云处理、网络分析、云原生工作流（STAC、COG、Planetary Computer），以及 8 种编程语言（Python、R、Julia、JavaScript、C++、Java、Go、Rust）和 500+ 代码示例。覆盖 30+ 科学领域。

**典型使用场景**：
- 遥感工作流
- GIS 分析
- 空间 ML
- 地球观测数据处理
- 水文建模

---

### 9. networkx（复杂网络分析）

**一句话简介**：创建、分析和可视化复杂网络和图的综合工具包。

**详细描述**：
NetworkX 支持四种图类型（Graph、DiGraph、MultiGraph、MultiDiGraph）。提供 100+ 算法（最短路径、中心性、聚类、社区检测、连通性、树算法、匹配、图着色、同构），50+ 图生成器（经典、随机、格子和专业网络），以及 8+ 布局算法（弹簧/力导向、圆形、谱、Kamada-Kawai）。支持多种 I/O 格式和交互式可视化。

**典型使用场景**：
- 社交网络分析
- 生物网络（PPI、基因调控网络）
- 交通系统
- 引文网络
- 知识图谱

---

### 10. exploratory-data-analysis（探索性数据分析）

**一句话简介**：具有自动统计、可视化和洞察的综合 EDA 工具包。

**详细描述**：
该技能为任何表格数据集提供自动化的探索性数据分析：生成描述性统计、分布可视化、缺失值分析、相关性矩阵、异常值检测和数据质量报告。帮助研究人员快速理解数据特征和发现模式。

**典型使用场景**：
- 新数据集的初步探索
- 数据质量评估
- 特征关系发现
- 假设生成

---

### 11. statistical-analysis（统计分析）

**一句话简介**：综合统计检验、功效分析和实验设计。

**详细描述**：
该技能提供假设检验、置信区间计算、功效分析、样本量估计和实验设计指导。涵盖参数和非参数检验、方差分析、回归诊断和多重比较校正。

**典型使用场景**：
- 实验设计
- 假设检验
- 功效分析
- 样本量计算
- 统计报告
