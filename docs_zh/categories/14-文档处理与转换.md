# 分类十四：文档处理与转换

> 本分类包含 **6 个技能**，覆盖 Word、Excel、PowerPoint、PDF 文档的创建/编辑/转换，以及多格式到 Markdown 的转换。

---

## 分类概述

文档处理与转换是科研工作的日常需求。本分类的技能使 AI Agent 能够创建和编辑 Office 文档（DOCX、XLSX、PPTX）、处理 PDF 文件，以及将 20+ 种文件格式转换为 LLM 友好的 Markdown。特别值得注意的是 MarkItDown 技能，它能够将几乎任何常见文档格式转换为结构化的 Markdown，为 RAG 系统和 LLM 处理提供了极大的便利。

---

## 技能清单

### 1. docx（Word 文档处理）

**一句话简介**：创建、读取、编辑和操作 Word 文档（.docx 文件）。

**详细描述**：
支持专业文档格式化：目录、标题、页码、信头、查找替换、修订痕迹、评论和图像插入。使用 python-docx 库实现 Word 文档的完整操作。

**典型使用场景**：
- 研究报告
- 备忘录
- 信件
- 模板创建
- 任何 Word 文档交付物

---

### 2. xlsx（Excel 电子表格）

**一句话简介**：电子表格创建、编辑和分析，支持公式、格式化、数据分析和可视化。

**详细描述**：
使用 openpyxl 库进行 Excel 文件的完整操作：单元格读写、公式计算、条件格式化、数据透视表、图表创建、样式设置和多工作表管理。支持将 pandas DataFrame 导出为格式化 Excel。

**典型使用场景**：
- 数据报告
- 财务分析
- 实验数据记录
- 自动报告生成
- 多工作表数据管理

---

### 3. pptx（PowerPoint 演示文稿）

**一句话简介**：创建、读取、编辑和操作 PowerPoint 演示文稿（.pptx 文件）。

**详细描述**：
使用 python-pptx 库进行幻灯片的完整操作：创建新幻灯片、编辑现有幻灯片、添加文本/图像/图表/表格、设置动画和过渡效果、合并或拆分演示文稿、处理模板和布局、演讲者备注和评论。

**典型使用场景**：
- 幻灯片制作
- 演讲稿
- 从演示文稿提取文本
- 编辑现有幻灯片
- 合并或拆分演示文稿

---

### 4. pdf（PDF 文件处理）

**一句话简介**：读取、创建、合并、拆分、旋转、水印、加密/解密和操作 PDF 文件。

**详细描述**：
支持文本/表格提取、表单填写、扫描 PDF 的 OCR、图像提取和格式转换。使用 PyPDF2、pdfplumber 和 pytesseract 等工具实现全面的 PDF 操作。

**典型使用场景**：
- 论文 PDF 文本提取
- 批量 PDF 合并/拆分
- 表单自动填写
- PDF 转其他格式
- 扫描文档 OCR

---

### 5. markitdown（多格式转 Markdown）

**一句话简介**：将 20+ 文件格式转换为针对 LLM 处理优化的 Markdown。

**详细描述**：
转换 Office 文档（PDF、DOCX、PPTX、XLSX）、带 OCR 的图像、带转录的音频、网页内容（HTML、YouTube 转录、EPUB）和结构化数据（CSV、JSON、XML），同时保留文档结构（标题、列表、表格、超链接）。特性包括 Azure Document Intelligence 集成增强 PDF 表格提取、GPT-4o LLM 图像描述、ZIP 批量处理、无临时文件的流式方法和插件系统。

**典型使用场景**：
- 为 RAG 系统准备文档
- PDF 和 Office 文件文本提取
- 音频转录
- 图像 OCR
- 文档分析流程
- LLM 训练数据准备

---

### 6. markdown-mermaid-writing（Markdown 与 Mermaid 图表）

**一句话简介**：创建科学文档、报告、分析和可视化的综合 Markdown 和 Mermaid 图表写作技能。

**详细描述**：
将文本图表确立为默认文档标准，提供完整风格指南、24 种图表类型参考和 9 个文档模板。支持流程图、时序图、类图、状态图、实体关系图、用户旅程图、甘特图、饼图和 Git 图等多种 Mermaid 图表类型。

**典型使用场景**：
- 技术文档编写
- 系统架构图
- 流程文档
- 数据流程图
- 项目计划可视化
