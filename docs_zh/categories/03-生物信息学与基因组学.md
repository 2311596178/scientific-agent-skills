# 分类三：生物信息学与基因组学

> 本分类包含 **19 个技能**，覆盖序列分析、单细胞组学、差异表达分析、基因组数据管理和高性能基因组计算等核心生物信息学领域。

---

## 分类概述

生物信息学与基因组学是 Scientific Agent Skills 中技能数量最多的领域。从 DNA/RNA/蛋白质序列的处理（BioPython），到单细胞 RNA 测序的端到端分析（Scanpy、scvi-tools），再到大规模基因组变异的存储和查询（TileDB-VCF），本分类提供了现代基因组学研究的完整工具链。特别值得关注的是，单细胞分析子领域拥有从数据存储格式（AnnData）、预处理（Scanpy）、深度建模（scvi-tools）到轨迹推断（scVelo）和基因调控网络重建（Arboreto）的全栈技能支持。

## 技能清单

### 1. anndata（注释数据矩阵）

**一句话简介**：专为单细胞基因组学设计的高效注释数据矩阵处理 Python 包。

**详细描述**：
AnnData 是单细胞分析生态系统的核心数据格式。基于 HDF5 的 h5ad 文件格式实现高效的 I/O 和压缩；与 pandas DataFrames 集成处理元数据；支持稀疏矩阵（scipy.sparse）以节省内存；分层数据组织（X 为主数据矩阵、obs 为观测注释、var 为变量注释、obsm/varm 为多维度注释、obsp/varp 为成对矩阵）。与 Scanpy、scvi-tools 等工具无缝集成，支持懒加载、分块操作和与其他格式（CSV、HDF5、Zarr）的转换。

**典型使用场景**：
- 单细胞 RNA-seq 数据的标准化存储
- 多模态单细胞数据（RNA+ATAC、CITE-seq）管理
- 空间转录组学数据处理
- 任何需要高效存储和操作的高维注释数据

### 2. arboreto（基因调控网络推断）

**一句话简介**：使用集成树方法从单细胞 RNA-seq 数据高效推断基因调控网络（GRN）。

**详细描述**：
Arboreto 实现了 GRNBoost2（基于梯度提升的网络推断）和 GENIE3（基于随机森林的推断）算法，针对大规模单细胞数据集进行了优化。支持并行处理、稀疏矩阵、与 Scanpy/AnnData 工作流集成、可自定义超参数，以及兼容网络分析工具的输出格式。提供潜在调控相互作用（转录因子-靶基因对）的排名列表及置信度评分。

**典型使用场景**：
- 从单细胞数据识别转录因子-靶基因关系
- 重建细胞类型特异性基因调控网络
- 理解细胞分化和发育中的调控程序
- 推断基因表达中的因果关系

### 3. biopython（生物信息学计算库）

**一句话简介**：用于计算生物学和生物信息学的综合 Python 库，提供序列操作、数据库访问和生物数据分析工具。

**详细描述**：
BioPython 是生物信息学领域最成熟的 Python 库之一。核心功能包括：序列对象（Seq、SeqRecord、SeqIO）支持 DNA/RNA/蛋白质序列和生物字母验证；文件格式解析器（FASTA、FASTQ、GenBank、EMBL、Swiss-Prot、PDB、SAM/BAM、VCF、GFF）；NCBI 数据库访问（Entrez Programming Utilities 用于 PubMed、GenBank、BLAST、分类学）；BLAST 集成；序列比对（成对和多重比对）；系统发育学（Bio.Phylo 构建和操作树）；群体遗传学和蛋白质结构分析。支持与 NumPy、pandas 集成。

**典型使用场景**：
- DNA/RNA/蛋白质序列分析和操作
- 批量下载和解析 GenBank/PubMed 数据
- BLAST 结果解析和自动化
- 系统发育树构建和可视化
- 多种生物信息学文件格式转换

### 4. bioservices（生物网络服务统一访问）

**一句话简介**：统一编程访问 40+ 生物网络服务和数据库的 Python 库。

**详细描述**：
BioServices 支持主要生物信息学资源，包括 KEGG（通路和化合物数据）、UniProt（蛋白质序列和注释）、ChEBI（化学实体）、ChEMBL（生物活性分子）、Reactome（通路）、IntAct（蛋白质相互作用）、BioModels（生物模型）等。特性包括：跨不同服务的一致 API、自动结果缓存、错误处理和重试逻辑、REST 和 SOAP Web 服务支持、结果转换为 Python 对象。处理认证、速率限制和 API 版本控制。

**典型使用场景**：
- 从多个生物数据库自动检索数据
- 构建集成多源数据的生物信息学流程
- 无需手动浏览网页即可程序化访问生物网络资源

### 5. cellxgene-census（CELLxGENE 单细胞数据普查）

**一句话简介**：查询和分析 CZ CELLxGENE Discover 普查中的大规模单细胞 RNA-seq 数据。

**详细描述**：
CELLxGENE Census 提供对 5000 万+ 细胞和 1000+ 数据集的访问，具有标准化注释和元数据。使用 TileDB-SOMA 格式实现可扩展查询，与 AnnData 和 Scanpy 集成进行下游分析。支持按细胞类型、组织、疾病或其他元数据过滤和查询，在下载前对数据集进行子集化，减少数据传输和内存需求。支持本地缓存和批处理操作。

**典型使用场景**：
- 大规模单细胞数据的跨数据集比较
- 构建参考数据集用于注释
- 细胞类型发现和验证
- 公共单细胞数据的探索性分析

### 6. deeptools（NGS 数据探索和可视化）

**一句话简介**：用于探索和可视化高通量测序数据的综合 Python 工具套件。

**详细描述**：
deepTools 特别针对 ChIP-seq、RNA-seq 和 ATAC-seq 实验设计。提供命令行工具和 Python API 处理 BAM 和 bigWig 文件。关键功能包括：质量控制指标（plotFingerprint、plotCorrelation）、覆盖度轨迹生成（bamCoverage 创建 bigWig）、热图矩阵生成（computeMatrix、plotHeatmap、plotProfile）、比较分析（multiBigwigSummary、plotPCA）。支持标准化方法、分箱选项和多种可视化输出。

**典型使用场景**：
- ChIP-seq 峰可视化
- RNA-seq 覆盖度分析
- ATAC-seq 信号轨迹
- 比较基因组学
- NGS 数据质量控制

### 7. flowio（流式细胞术标准文件处理）

**一句话简介**：读取和操作流式细胞术标准（FCS）文件的 Python 库。

**详细描述**：
FlowIO 支持 FCS 文件（版本 2.0、3.0、3.1）的高效解析，访问事件数据（荧光强度、散射参数）、元数据提取（关键词、参数、采集设置），以及转换为 pandas DataFrames 或 NumPy 数组。支持大 FCS 文件、多数据段处理、文本段和分析段访问。

**典型使用场景**：
- 流式细胞术数据分析
- 高通量筛选
- 免疫细胞分型
- FCS 文件的自动化批量处理

### 8. gget（基因组数据库快速查询）

**一句话简介**：简单统一界面的基因组数据库高效查询命令行工具和 Python 包。

**详细描述**：
gget 提供对 Ensembl（基因信息、序列、直系同源、变异）、UniProt（蛋白质序列和注释）、NCBI（BLAST 搜索、基因信息）、PDB（蛋白质结构）、COSMIC（癌症突变）等数据库的快速访问。支持单命令查询、自动结果格式化、批处理查询、pandas DataFrame 集成，以及命令行和 Python API。

**典型使用场景**：
- 快速基因查询和序列检索
- 变异注释
- 蛋白质结构访问
- 无需复杂 API 设置的快速数据库查询

### 9. geniml（基因组区间机器学习工具包）

**一句话简介**：在 BED 文件上构建 ML 模型的无监督方法工具包。

**详细描述**：
geniml 的关键能力包括：Region2Vec（基因组区间的 word2vec 风格嵌入）、BEDspace（区域和元数据标签的联合嵌入）、scEmbed（单细胞 ATAC-seq 数据的细胞级嵌入）、consensus peak 构建（四种统计方法），以及综合工具（BBClient、BEDshift、Text2BedNN）。属于 BEDbase 生态系统。支持预训练模型和 Hugging Face 集成。

**典型使用场景**：
- 区域相似性搜索
- 染色质可及性数据的降维
- scATAC-seq 聚类和细胞类型注释
- 基因组区间特征向量的 ML 任务

### 10. gtars（高性能基因组区间分析工具包）

**一句话简介**：用于基因组区间分析的高性能 Rust 工具包，提供 Python 绑定。

**详细描述**：
gtars 提供 IGD 索引的高效重叠检测、覆盖度轨迹生成（WIG/BigWig 格式的 uniwig 模块）、基因组标记化（TreeTokenizer）、参考序列管理（refget 协议）、单细胞基因组学的片段处理，以及片段评分。关键模块包括：标记化器、重叠计算器、uniwig（ATAC-seq/ChIP-seq/RNA-seq 覆盖度）、refget、bbcache、评分和 fragsplit。支持并行处理、内存映射文件和大数据集的流式处理。

**典型使用场景**：
- 基因组 ML 预处理
- 调控元件分析
- 变异注释
- 染色质可及性分析

Polars-Bio 提供 BED/VCF/BAM/GFF 区间的重叠、最近邻、合并、覆盖度、补集和减法操作。流式和云原生架构支持大规模基因组分析，是 bioframe 的更快替代方案。

**典型使用场景**：
- 大规模基因组区间操作

- 高性能基因组数据处理流程

### 12. pysam（基因组数据文件读写）

**一句话简介**：读取、写入和操作基因组数据文件（SAM/BAM/CRAM、VCF/BCF、FASTA/FASTQ）。

**详细描述**：
pysam 是处理基因组学标准格式的核心 Python 库。支持 SAM/BAM/CRAM 比对文件的读写、VCF/BCF 变异文件的解析、FASTA/FASTQ 序列操作，以及 pileup 分析、覆盖度计算和生物信息学工作流。

**典型使用场景**：
- 比对文件的过滤和统计
- 变异调用的后处理
- 参考序列的随机访问
- 基因组数据的批量处理

### 13. pydeseq2（差异基因表达分析）

**一句话简介**：用于 bulk RNA-seq 数据的 DESeq2 差异基因表达分析方法的 Python 实现。

**详细描述**：
PyDESeq2 使用负二项广义线性模型确定实验条件间的差异表达。关键功能包括：文库大小归一化的 size factor 估计、离散度估计和收缩、Wald 检验或似然比检验的假设检验、多重检验校正（Benjamini-Hochberg FDR）、结果过滤和排序。处理复杂实验设计、批次效应和重复。产生每个基因的 fold-change 估计、p 值和校正 p 值。

**典型使用场景**：
- 鉴定不同条件下的差异表达基因
- RNA-seq 实验分析
- 生物标志物发现
- 需要严格统计分析的基因表达研究

### 14. scanpy（单细胞 RNA 测序分析）

**一句话简介**：基于 AnnData 的单细胞 RNA-seq 数据分析综合 Python 工具包。

**详细描述**：
Scanpy 是单细胞分析领域最广泛使用的工具之一。提供端到端工作流：预处理（质量控制、标准化、对数转换）、降维（PCA、UMAP、t-SNE、ForceAtlas2）、聚类（Leiden、Louvain、层次聚类）、marker 基因鉴定、轨迹推断（PAGA、扩散图）和可视化。高效处理大规模数据集（数百万细胞），支持多模态数据（RNA+ATAC、CITE-seq）、批次校正方法和发表级绘图函数。支持部分操作的 GPU 加速。

**典型使用场景**：
- 单细胞 RNA-seq 数据的端到端分析
- 细胞类型鉴定和注释
- 发育轨迹分析
- 批次效应校正
- 单细胞基因组学综合工作流

### 15. scvelo（RNA 速度分析）

**一句话简介**：从未剪接/剪接 mRNA 动态估计细胞状态转变的 RNA 速度分析。

**详细描述**：
scVelo 推断轨迹方向、计算潜在时间，并识别单细胞 RNA-seq 数据中的驱动基因。作为 Scanpy/scVI-tools 轨迹推断的补充，支持在单细胞分辨率下研究细胞分化动态和谱系决策。

**典型使用场景**：
- 细胞分化动态研究
- 发育轨迹方向推断
- 驱动基因识别
- 谱系决策分析

### 16. scvi-tools（单细胞组学深度学习）

**一句话简介**：用于单细胞组学分析的概率深度学习模型框架。

**详细描述**：
scvi-tools 基于 PyTorch 提供 25+ 模型：scVI/scANVI（RNA-seq 整合和细胞类型注释）、totalVI（CITE-seq 蛋白质+RNA）、MultiVI（多组学 RNA+ATAC 整合）、PeakVI（ATAC-seq 分析）、DestVI/Stereoscope/Tangram（空间转录组学反卷积）、MethylVI（甲基化）、CytoVI（流式/质谱细胞术）、VeloVI（RNA 速度）、contrastiveVI（扰动研究）、Solo（双细胞检测）。支持 GPU 加速、参考映射（scArches）和概率差异表达。

**典型使用场景**：
- 大规模单细胞数据整合
- 多模态单细胞分析
- 空间转录组学反卷积
- 批次校正和细胞类型注释
- 双细胞检测

### 17. scikit-bio（生物信息学 Python 库）

**一句话简介**：提供生物序列分析数据结构、算法和解析器的生物信息学 Python 库。

**详细描述**：
基于 NumPy、SciPy 和 pandas 构建。核心功能包括：序列对象（DNA、RNA、蛋白质序列）和生物字母验证、序列比对算法（局部、全局、半全局）、系统发育树操作、多样性指标（alpha、beta、系统发育多样性）、距离度量、文件格式解析器（FASTA、FASTQ、QIIME 格式、Newick）和统计分析工具。提供 scikit-learn 兼容的转换器。

**典型使用场景**：
- 微生物生态学（16S rRNA 分析）
- 宏基因组学
- 系统发育分析
- 群落多样性计算

TileDB-VCF 支持可扩展的 VCF/BCF 导入（增量样本添加）、压缩存储、跨基因组区域和样本的并行查询，以及群体基因组学工作流的导出能力。支持内存高效查询、云存储集成（S3、Azure、GCS）和 CLI 工具。可扩展到企业级分析。

### 19. zarr-python（分块压缩 N 维数组存储）

**一句话简介**：实现 Zarr 分块压缩 N 维数组存储格式的 Python 库。

**详细描述**：
Zarr 支持 NumPy 风格数组的分块存储、多种压缩编解码器（zlib、blosc、lz4、zstd）、部分数组读取（仅加载所需块）、本地文件系统和云存储（S3、GCS、Azure）支持，以及与 NumPy、Dask、Xarray 的集成。支持并行读写操作，针对云存储后端优化。

**典型使用场景**：
- 大规模科学数据存储
- 基于云的数组存储
- 核外数组操作
- 基因组学、影像、气候数据的多维数据集高效存储
