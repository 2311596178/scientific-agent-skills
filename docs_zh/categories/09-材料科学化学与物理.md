# 分类九：材料科学、化学与物理

> 本分类包含 **3 个技能**，覆盖天文数据分析、代谢网络建模和材料计算分析。

---

## 分类概述

材料科学、化学与物理是连接实验科学与计算科学的重要桥梁。本分类的技能涵盖从天体物理坐标变换（Astropy）到基因组规模代谢网络分析（COBRApy），再到晶体结构处理和相图构建（Pymatgen），为物理科学和材料研究提供了强大的计算支持。

---

## 技能清单

### 1. astropy（天文数据分析）

**一句话简介**：用于天文学和天体物理学研究的综合 Python 库。

**详细描述**：
Astropy 是天文学研究的核心 Python 库。提供坐标系统转换（ICRS、Galactic、FK5、AltAz）、物理单位和量（自动维度一致性）、FITS 文件操作（读写、操作头部和数据）、宇宙学计算（光度距离、回望时间、哈勃参数）、精确时间处理（UTC、TAI、TT、TDB 和 JD、MJD、ISO 格式）、表格操作（FITS、CSV、HDF5、VOTable）、WCS 像素-世界坐标转换、天文常数、建模框架和统计函数。

**典型使用场景**：
- 天体坐标转换
- FITS 图像/表格处理
- 宇宙学距离计算
- 质心时间校正
- 星表交叉匹配
- 天文数据分析

---

### 2. cobrapy（代谢网络约束分析）

**一句话简介**：用于代谢网络的约束重建和分析（COBRA）的 Python 包。

**详细描述**：
COBRApy 提供通量平衡分析（FBA）预测最优代谢通量、通量变异性分析（FVA）、基因敲除模拟、通路分析、模型验证，以及与其他 COBRA Toolbox 格式（SBML、JSON）的集成。支持多种优化目标（生物量产率、ATP 产生、代谢物产生）、约束处理（反应边界、基因-蛋白质-反应关联）和模型比较。

**典型使用场景**：
- 代谢工程
- 系统生物学
- 生物技术应用
- 细胞代谢理解
- 代谢表型预测

---

### 3. pymatgen（材料基因组学）

**一句话简介**：用于材料科学计算和分析的 Python Materials Genomics 库。

**详细描述**：
Pymatgen 提供晶体结构操作、相图构建、电子结构分析和材料性质计算。特性包括：对称分析的结构对象、空间群确定、结构匹配和比较、基于生成能的相图生成、能带结构和态密度分析、缺陷计算、表面和界面分析，以及与 DFT 代码（VASP、Quantum ESPRESSO、ABINIT）的集成。支持 Materials Project 数据库集成和结构文件 I/O（CIF、POSCAR、VASP）。

**典型使用场景**：
- 材料发现
- 晶体结构分析
- 相稳定性预测
- 电子结构计算
- 计算材料科学
