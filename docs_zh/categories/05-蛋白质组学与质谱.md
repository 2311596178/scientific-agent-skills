# 分类五：蛋白质组学与质谱分析

> 本分类包含 **2 个技能**，覆盖质谱数据的处理、相似性匹配和蛋白质组学/代谢组学分析。

---

## 分类概述

蛋白质组学和质谱分析是系统生物学研究的核心技术。本分类的技能支持从原始质谱数据的处理到肽段鉴定和定量分析的完整流程。matchms 专注于质谱数据的相似性匹配和谱库搜索，而 pyOpenMS 提供了更全面的 LC-MS/MS 数据处理、特征检测和蛋白质鉴定能力。

---

## 技能清单

### 1. matchms（质谱数据处理与相似性匹配）

**一句话简介**：支持 40+ 过滤器的质谱数据处理和相似性匹配库，提供谱库匹配和分子指纹比较。

**详细描述**：
matchms 是质谱数据处理的核心工具，特别适用于代谢组学和天然产物研究。提供 40+ 过滤器用于质谱数据清洗和预处理、谱库匹配算法（Cosine、Modified Cosine、Neutral Losses）、元数据协调、分子指纹比较，以及对多种文件格式的支持（MGF、MSP、mzML、JSON）。可以基于质谱相似性识别未知化合物、构建质谱数据库和进行分子网络分析。

**典型使用场景**：
- 非靶向代谢组学的化合物鉴定
- 基于质谱相似性的分子网络构建
- 谱库搜索和匹配
- 代谢物注释
- 天然产物发现

---

### 2. pyopenms（全面质谱数据分析）

**一句话简介**：用于蛋白质组学和代谢组学的全面质谱数据分析库。

**详细描述**：
pyOpenMS 是 OpenMS 的 Python 绑定，是蛋白质组学和代谢组学领域最全面的开源质谱数据处理平台之一。提供 LC-MS/MS 数据处理、肽段鉴定、特征检测、定量分析、化学计算，以及与 Comet、Mascot、MSGF+ 等搜索引擎的集成。支持从原始数据到蛋白质鉴定的完整流程，包括保留时间对齐、峰检测、同位素模式识别和统计分析。

**典型使用场景**：
- 蛋白质组学数据的自下而上分析
- LC-MS 代谢组学数据处理
- 肽段和蛋白质鉴定
- 定量蛋白质组学（标记和非标记）
- 代谢物特征检测和注释
