# 分类八：机器学习与人工智能

> 本分类包含 **18 个技能**，覆盖深度学习、强化学习、时间序列分析、贝叶斯方法、量子机器学习、图神经网络和模型可解释性等前沿 AI 领域。

---

## 分类概述

机器学习与人工智能是 Scientific Agent Skills 中技能数量最多的分类之一。从经典的 scikit-learn 到前沿的量子计算（Qiskit、PennyLane、Cirq、QuTiP），从时间序列基础模型（TimesFM）到图神经网络（Torch Geometric），从强化学习（Stable Baselines3、PufferLib）到贝叶斯统计（PyMC），本分类提供了极为丰富的 AI 工具链。特别值得关注的是量子机器学习子领域拥有 4 个互补的技能：Qiskit（IBM 量子硬件）、PennyLane（跨平台自动微分）、Cirq（Google 量子硬件）和 QuTiP（量子力学模拟）。

## 技能清单

### 1. scikit-learn（经典机器学习）

**一句话简介**：经典机器学习的行业标准 Python 库，提供全面的监督学习、无监督学习和数据预处理工具。

**详细描述**：
scikit-learn 是数据科学领域使用最广泛的 ML 库。监督学习包括分类（Logistic Regression、SVM、决策树、随机森林、梯度提升、朴素贝叶斯、KNN、MLP）和回归（线性、岭、Lasso、ElasticNet、SVR）。无监督学习包括聚类（K-Means、DBSCAN、HDBSCAN、谱聚类、GMM）、降维（PCA、t-SNE、Isomap、NMF）和异常检测（IsolationForest、OneClassSVM）。数据预处理包括缩放、编码、插补和特征工程。模型评估提供交叉验证、网格搜索和 30+ 评估指标。一致的 fit/predict/transform API 和与 NumPy/pandas 的集成。

**典型使用场景**：
- 预测建模
- 客户细分
- 异常检测
- 特征工程
- 医学诊断
- 欺诈检测
- 推荐系统

### 2. pytorch-lightning（深度学习框架）

**一句话简介**：组织 PyTorch 代码以消除样板代码，同时保持完整灵活性的深度学习框架。

**详细描述**：
PyTorch Lightning 自动化训练工作流（epoch/批次迭代、优化器步骤、梯度管理、检查点）、支持多 GPU/TPU 训练（DDP/FSDP/DeepSpeed 策略）。包含 LightningModule（模型组织）、Trainer（自动化）、LightningDataModule（数据流程）、callbacks（可扩展性），以及与 TensorBoard、Wandb、MLflow 的实验追踪集成。

**典型使用场景**：
- 大规模深度学习模型训练
- 多 GPU 分布式训练
- 生产级模型部署
- 减少 PyTorch 样板代码

### 3. transformers（预训练模型库）

**一句话简介**：用于 NLP、计算机视觉、音频和多模态任务的最先进机器学习模型库。

**详细描述**：
Hugging Face Transformers 提供 100 万+ 预训练模型，可通过 pipeline 访问（文本分类、NER、QA、摘要、翻译、文本生成、图像分类、目标检测、ASR、VQA）。全面的 Trainer API 支持分布式训练和混合精度，灵活的文本生成支持多种解码策略（贪婪、束搜索、采样）。Auto 类自动选择架构（BERT、GPT、T5、ViT、BART 等）。

**典型使用场景**：
- 自然语言处理任务
- 计算机视觉模型微调
- 多模态学习
- 大规模预训练模型部署

---

### 4. torch-geometric（图神经网络）

**一句话简介**：用于分子和几何数据的图神经网络库。

**详细描述**：
Torch Geometric 提供消息传递神经网络、图注意力网络、图卷积网络和多种图池化操作的实现。特别适用于分子图、社交网络和点云数据的分析。与 PyTorch 完全兼容，支持大规模图数据的 mini-batch 处理。

**典型使用场景**：
- 分子图表示学习
- 蛋白质-蛋白质相互作用网络分析
- 点云数据处理
- 社交网络分析

### 5. stable-baselines3（强化学习）

**一句话简介**：基于 PyTorch 的强化学习库，提供 PPO、SAC、DQN 等可靠实现。

**详细描述**：
Stable Baselines3 提供 PPO、SAC、DQN、TD3、DDPG、A2C、HER、RecurrentPPO 等算法的可靠实现。支持标准或自定义 Gymnasium 环境的 RL 智能体训练、回调监控、向量化环境并行训练、多输入策略（字典观测）和 HER 目标条件学习。

**典型使用场景**：
- 游戏 AI 开发
- 机器人控制策略学习
- 资源调度优化
- 序列决策问题

### 6. pufferlib（高性能强化学习）

**一句话简介**：通过优化向量化和原生多智能体支持实现 100 万-400 万步/秒的高性能强化学习库。

**详细描述**：
PufferLib 包含 PuffeRL 高效 PPO 训练、20+ Ocean 环境、自动空间扁平化的无缝框架集成、共享内存缓冲区的零拷贝向量化、分布式训练支持，以及训练工作流、环境开发、向量化优化、策略架构和第三方集成的综合参考指南。

**典型使用场景**：
- 大规模 RL 训练
- 自定义环境开发
- 多智能体系统
- 并行模拟优化

### 7. pymc（贝叶斯统计建模）

**一句话简介**：用于贝叶斯统计建模和概率编程的综合 Python 库。

**详细描述**：
PyMC 提供直观的概率模型构建语法、先进的 MCMC 采样算法（NUTS、Metropolis-Hastings、Slice）、变分推断方法（ADVI、SVGD）、高斯过程、时间序列模型（ARIMA、状态空间模型）和模型比较工具（WAIC、LOO）。支持自动微分、GPU 加速、并行采样和与 ArviZ 的集成。支持层次模型、混合模型、生存分析和自定义分布。

**典型使用场景**：
- 贝叶斯数据分析
- 不确定性量化
- A/B 测试
- 时间序列预测
- 层次建模

### 8. statsmodels（统计建模与计量经济学）

**一句话简介**：统计建模和计量经济学库（OLS、GLM、logit/probit、ARIMA、时间序列预测、假设检验、诊断）。

**详细描述**：
statsmodels 提供经典统计模型的完整实现：线性回归、广义线性模型、离散选择模型（logit、probit、MNLogit）、时间序列分析（ARIMA、SARIMAX、VAR、状态空间模型）、假设检验、统计诊断和协整检验。特别适用于经济学、社会科学和生物医学研究中的统计推断。

**典型使用场景**：
- 回归分析和统计推断
- 时间序列建模
- 假设检验
- 面板数据分析
- 因果推断

### 9. shap（模型可解释性）

**一句话简介**：使用博弈论中的 Shapley 值进行模型可解释性和解释的统一方法。

**详细描述**：
SHAP 提供 TreeExplainer（XGBoost/LightGBM/随机森林的快速精确解释）、DeepExplainer（TensorFlow/PyTorch 神经网络）、KernelExplainer（模型无关）和 LinearExplainer。包含全面的可视化（瀑布图、蜂群图、散点图、条形图/力图/热图），支持模型调试、公平性分析、特征工程指导和生产部署。

**典型使用场景**：
- 解释模型预测
- 特征重要性分析
- 模型调试
- 公平性评估
- 监管合规的模型透明性

---

### 10. aeon（时间序列机器学习）

**一句话简介**：提供跨 7 个领域最先进算法的全面 scikit-learn 兼容时间序列机器学习工具包。

**详细描述**：
aeon 涵盖分类（13 类算法：ROCKET 变体、深度学习 InceptionTime/ResNet/FCN、基于距离的 DTW/ERP/LCSS、基于 shapelet、字典方法 BOSS/WEASEL、混合集成 HIVECOTE）、回归（9 类）、聚类、预测（ARIMA、ETS、Theta、TCN、DeepAR）、异常检测、分割和相似性搜索。包含 40+ 距离度量、大量变换（ROCKET/MiniRocket/MultiRocket、Catch22/TSFresh、SAX/PAA）、20+ 深度学习架构和 UCR/UEA 基准测试工具。

**典型使用场景**：
- 生理信号分类（ECG、EEG）
- 工业传感器监测
- 金融预测
- 变化点检测
- 活动识别
- 预测性维护

### 11. timesfm-forecasting（时间序列基础模型预测）

**一句话简介**：使用 Google TimesFM 基础模型进行零样本时间序列预测。

**详细描述**：
TimesFM 是 Google 开发的零样本时间序列基础模型，无需训练自定义模型即可对任何单变量时间序列（销售、传感器、能源、生命体征、天气）进行预测。支持 CSV/DataFrame/array 输入，提供点预测和预测区间。包含预飞行系统检查脚本，在首次使用前验证 RAM/GPU。

**典型使用场景**：
- 快速时间序列预测原型
- 无需训练数据的基线预测
- 多领域时间序列分析

### 12. scikit-survival（生存分析）

**一句话简介**：用于审查数据生存分析和事件时间建模。

**详细描述**：
基于 scikit-learn 构建，提供 Cox 比例风险模型（CoxPHSurvivalAnalysis、带弹性网络的 CoxnetSurvivalAnalysis）、集成方法（随机生存森林、梯度提升）、生存支持向量机、非参数估计器（Kaplan-Meier、Nelson-Aalen）、竞争风险分析和专门的评估指标（一致性指数、时间依赖 AUC、Brier 分数）。处理右审查数据，支持 scikit-learn 流程的特征选择和超参数调优。

**典型使用场景**：
- 临床预后预测
- 客户流失分析
- 设备故障预测
- 生物医学生存分析

---

### 13. umap-learn（降维与流形学习）

**一句话简介**：用于降维和流形学习的均匀流形近似和投影（UMAP）Python 实现。

**详细描述**：
UMAP 提供快速、可扩展的非线性降维，同时保留高维数据的局部和全局结构。支持监督和非监督降维、混合数据类型、scikit-learn API 集成，以及 numba 加速的高效实现。产生的低维嵌入（通常为 2D 或 3D）适合可视化和下游分析。在保留全局结构方面通常优于 t-SNE。

**典型使用场景**：
- 高维数据可视化
- 特征提取
- 单细胞数据降维
- 聚类预处理

### 14. pymoo（多目标优化）

**一句话简介**：使用进化算法进行多目标优化的 Python 框架。

**详细描述**：
PyMOO 实现 NSGA-II、NSGA-III、MOEA/D、SPEA2 等最先进算法。支持约束和无约束优化、连续/离散/混合变量问题、性能指标（超体积、IGD、GD）、Pareto 前沿可视化和并行评估。支持自定义问题定义和结果分析。

**典型使用场景**：
- 工程设计优化
- 药物分子多性质优化
- 超参数多目标调优
- 资源分配问题

### 15. qiskit（量子计算框架）

**一句话简介**：世界上最流行的开源量子计算框架，下载量超 1300 万，74% 开发者首选。

**详细描述**：
Qiskit 提供 100+ 量子门、83 倍更快的电路优化、采样器和估计器原语、可视化工具（电路图、直方图、Bloch 球）、后端无关执行（本地模拟器、IBM 量子硬件、IonQ、Amazon Braket）、错误缓解（ZNE、PEC）和算法库（Qiskit Nature 量子化学、Qiskit Optimization 组合优化、Qiskit Machine Learning 量子核）。支持脉冲级控制和 IBM Quantum Runtime 云执行。

**典型使用场景**：
- 变分量子本征求解器
- QAOA 组合优化
- 量子化学计算
- 量子机器学习
- 量子算法原型

### 16. pennylane（量子机器学习）

**一句话简介**：跨平台量子计算、量子机器学习和量子化学 Python 库。

**详细描述**：
PennyLane 支持 100+ 量子门、QNodes（带自动微分的量子函数）、与 PyTorch/JAX/NumPy 的无缝集成、跨模拟器和量子硬件（IBM、Amazon Braket、Google、Rigetti、IonQ）的设备无关执行。量子化学模块包括分子哈密顿量构建、VQE 基态能量和可微分 Hartree-Fock 求解器。支持 VQE、QAOA、量子神经网络、混合量子-经典模型和脉冲级编程。

**典型使用场景**：
- 变分量子本征求解器
- 量子电路机器学习
- 混合量子-经典神经网络
- 量子化学计算

### 17. cirq（Google 量子计算框架）

**一句话简介**：Google 量子计算框架，用于设计、模拟和运行量子电路。

**详细描述**：
Cirq 最适合针对 Google Quantum AI 硬件、设计噪声感知电路和运行量子表征实验。提供低级电路设计、噪声建模和硬件特定优化。与 IBM 硬件使用 Qiskit，量子 ML 自动微分使用 PennyLane，物理模拟使用 QuTiP。

**典型使用场景**：
- Google 量子硬件编程
- 噪声感知量子电路设计
- 量子表征实验
- 量子纠错研究

### 18. qutip（量子工具箱）

**一句话简介**：用于模拟和分析量子力学系统的 Python 量子工具箱。

**详细描述**：
QuTiP 提供量子态（ket、bra、密度矩阵）、量子算符（产生/湮灭算符、Pauli 矩阵）、时间演化求解器（薛定谔方程、Lindblad 主方程、量子轨迹蒙特卡洛、Floquet 方法）、分析工具（期望值、熵、保真度、纠缠、相关函数）和可视化（Bloch 球动画、Wigner 函数）。支持张量积、部分迹、时变哈密顿量和并行处理。

**典型使用场景**：
- 量子光学模拟
- 量子计算门操作
- 开放量子系统
- 量子信息理论
- 凝聚态物理
