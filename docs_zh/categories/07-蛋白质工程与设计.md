# 分类七：蛋白质工程与设计

> 本分类包含 **4 个技能**，覆盖蛋白质语言模型、糖工程、分子动力学模拟和自动化蛋白质测试验证。

---

## 分类概述

蛋白质工程与设计是连接计算生物学和实验生物学的关键领域。本分类的技能从序列层面（ESM 蛋白质语言模型）到结构层面（分子动力学），再到翻译后修饰（糖工程）和实验验证（Adaptyv 云平台），提供了蛋白质工程的全链条支持。特别值得注意的是 ESM3 作为多模态生成式蛋白质语言模型，能够同时推理序列、结构和功能。

---

## 技能清单

### 1. esm（进化尺度建模）

**一句话简介**：来自 EvolutionaryScale 的最先进蛋白质语言模型，用于蛋白质设计、结构预测和表示学习。

**详细描述**：
ESM 技能涵盖 ESM3（14 亿-980 亿参数的多模态生成模型，支持序列、结构和功能的同步推理，包含链式生成、逆折叠和功能条件设计）和 ESM C（3 亿-60 亿参数的高效嵌入模型，比 ESM2 快 3 倍，用于相似性分析和分类）。支持使用开放权重进行本地推理，以及通过 Forge API 进行可扩展的批量处理云端推理。

**典型使用场景**：
- 新型蛋白质设计
- 序列到结构的预测
- 结构到序列的设计
- 蛋白质嵌入和表示学习
- 功能注释
- 变体生成
- 定向进化工作流

---

### 2. glycoengineering（糖工程）

**一句话简介**：分析和工程蛋白质糖基化，扫描 N-糖基化序列并预测 O-糖基化热点。

**详细描述**：
该技能提供对蛋白质糖基化的计算分析能力：扫描序列中的 N-糖基化序列（N-X-S/T）、预测 O-糖基化热点，以及访问精选的糖工程工具（NetOGlyc、GlycoShield、GlycoWorkbench）。用于糖蛋白工程、治疗性抗体优化和疫苗设计。

**典型使用场景**：
- 治疗性抗体的糖基化优化
- 糖蛋白工程
- 疫苗设计中糖基化位点的评估
- 蛋白质糖基化模式的预测和分析

---

### 3. molecular-dynamics（分子动力学模拟）

**一句话简介**：使用 OpenMM 和 MDAnalysis 运行和分析分子动力学模拟。

**详细描述**：
该技能支持蛋白质/小分子系统的设置、力场定义、能量最小化和生产 MD 运行，以及轨迹分析（RMSD、RMSF、接触图、自由能面）。用于结构生物学、药物结合研究和生物物理研究。OpenMM 提供高性能 GPU 加速的分子动力学引擎，MDAnalysis 提供全面的轨迹分析工具。

**典型使用场景**：
- 蛋白质构象动态研究
- 药物-靶点结合机制分析
- 蛋白质稳定性评估
- 配体结合自由能计算
- 膜蛋白动力学

---

### 4. adaptyv（自动化蛋白质测试验证平台）

**一句话简介**：用于自动化蛋白质测试和验证的云实验室平台。

**详细描述**：
Adaptyv 支持通过 API 或 Web 界面提交蛋白质序列，约 21 天内接收实验结果。支持多种检测类型：结合检测（生物层干涉测量、KD/kon/koff 测量）、表达测试（大肠杆菌、哺乳动物、酵母或昆虫细胞中的蛋白质表达水平定量）、热稳定性测量（DSF 和 CD 测定 Tm）和酶活性检测。计算优化工具包括：NetSolP/SoluPt 溶解度预测、SolubleMPNN 序列重设计、ESM 序列似然评分、ipTM（AlphaFold-Multimer）界面稳定性评估和 pSAE 聚集风险量化。

**典型使用场景**：
- 抗体亲和力成熟
- 治疗性蛋白质可开发性评估
- 酶工程和优化
- 蛋白质稳定性改善
- AI 驱动蛋白质设计的验证
- 库筛选的表达和功能
- 先导优化与实验反馈
- 计算设计与湿实验验证的迭代循环
