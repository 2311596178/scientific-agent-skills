# 分类四：化学信息学与药物发现

> 本分类包含 **9 个技能**，覆盖分子操作、分子动力学、深度学习分子建模、虚拟筛选、分子对接和药物性质预测等药物研发全流程。

---

## 分类概述

化学信息学与药物发现是 Scientific Agent Skills 的核心优势领域之一。从基础的分子解析和描述符计算（RDKit），到先进的深度学习分子生成（DeepChem、TorchDrug）、扩散模型分子对接（DiffDock），再到云计算量子化学（Rowan），本分类提供了从苗头化合物发现到先导化合物优化的完整计算工具链。特别值得注意的是，技能之间形成了清晰的上下游关系：RDKit 用于基础分子操作 → datamol/molfeat 用于特征化和预处理 → DeepChem/TorchDrug 用于预测建模 → DiffDock 用于结构验证 → MedChem 用于成药性评估。

---

## 技能清单

### 1. rdkit（化学信息学工具包）

**一句话简介**：用于分子信息学和药物发现的开源化学信息学工具包。

**详细描述**：
RDKit 是化学信息学领域最广泛使用的开源库之一，拥有高性能 C++ 核心和 Python 绑定。提供全面的功能：分子 I/O（读写 SMILES、SDF、MOL、PDB 文件）、200+ 2D 和 3D 分子描述符、分子指纹（Morgan、RDKit、MACCS、拓扑扭转）、SMARTS 模式匹配的子结构搜索、分子对齐和 3D 坐标生成、药效团感知、反应处理和分子绘图。在制药工业和学术研究中广泛使用。

**典型使用场景**：
- 分子性质计算和筛选
- 虚拟筛选和分子相似性搜索
- 子结构匹配和 SMARTS 查询
- 分子可视化和结构生成
- 化学信息学工作流的基础操作

---

### 2. datamol（分子操作与特征化）

**一句话简介**：基于 RDKit 构建的分子操作和特征化 Python 库，提供增强的工作流和性能优化。

**详细描述**：
Datamol 在 RDKit 之上提供了更高级的抽象：分子 I/O（读写 SMILES、SDF、MOL 文件）、分子标准化和消毒、分子变换（互变异构体枚举、立体异构体生成）、分子特征化（描述符、指纹、图表示）、大数据集的并行处理，以及与机器学习流程的集成。特性包括优化的 RDKit 操作、重复计算缓存、分子过滤和预处理、pandas DataFrame 无缝集成。

**典型使用场景**：
- ML 模型的分子预处理
- 化合物库管理
- 分子相似性搜索
- 化学信息学数据流程

---

### 3. molfeat（分子特征化综合库）

**一句话简介**：提供 100+ 分子特征化器，将分子转换为适合机器学习的数值表示。

**详细描述**：
Molfeat 包括分子指纹（ECFP、MACCS、RDKit、Pharmacophore）、分子描述符（2D/3D、组成、拓扑、电子）、图表示（分子图、线图）和预训练模型（MolBERT、ChemBERTa、Uni-Mol 嵌入）。跨不同特征化器类型的统一 API、性能缓存、并行处理，以及与 scikit-learn、PyTorch、TensorFlow 的集成。支持传统化学信息学描述符和现代学习表示。

**典型使用场景**：
- 分子性质预测的输入准备
- 虚拟筛选的特征工程
- 分子相似性搜索
- 为 ML 模型准备分子数据

---

### 4. deepchem（分子机器学习深度学习框架）

**一句话简介**：基于 TensorFlow 和 PyTorch 的分子机器学习和药物发现深度学习框架。

**详细描述**：
DeepChem 提供图神经网络（GCN、GAT、MPNN、AttentiveFP）实现、分子特征化、预训练模型和 MoleculeNet 基准套件（50+ 数据集）。支持双后端（TensorFlow 和 PyTorch）、分布式训练、超参数优化、模型解释工具，以及与 RDKit 的集成。包含量子化学、毒性预测、ADMET 性质和结合亲和力预测的数据集。

**典型使用场景**：
- 分子性质预测
- 药物发现中的 ADMET 预测
- 毒性筛选
- 分子机器学习研究

---

### 5. torchdrug（药物发现 PyTorch 平台）

**一句话简介**：基于 PyTorch 的药物发现机器学习平台，提供 40+ 数据集和 20+ GNN 模型。

**详细描述**：
TorchDrug 涵盖分子性质预测、蛋白质建模、知识图谱推理、分子生成和逆合成规划。提供标准化的数据加载器、预定义的任务和评估指标，以及模块化的模型组件。支持快速原型开发和模型比较。

**典型使用场景**：
- 分子性质预测模型开发
- 蛋白质-配体相互作用建模
- 分子生成和优化
- 逆合成规划

---

### 6. diffdock（扩散模型分子对接）

**一句话简介**：基于扩散模型的最先进分子对接方法，预测蛋白质-配体结合构象和结合亲和力。

**详细描述**：
DiffDock 使用扩散模型生成多样化、高质量的结合构象，无需穷举搜索。与传统对接方法相比推理速度更快，生成多个多样化构象，提供预测置信度评分，支持柔性配体对接。在标准基准（PDBbind、CASF）上优于传统对接方法。

**典型使用场景**：
- 虚拟筛选
- 先导优化
- 结合构象预测
- 基于结构的药物设计
- 为更昂贵的方法生成初始构象

---

### 7. medchem（药物化学分析）

**一句话简介**：用于药物化学分析和类药性评估的 Python 库。

**详细描述**：
MedChem 提供分子描述符计算、ADMET（吸收、分布、代谢、排泄、毒性）性质预测、类药性过滤器（Lipinski 五规则、Veber 规则、Egan 规则、Muegge 规则）、分子复杂度指标和合成可及性评分。与 RDKit 集成，支持大数据集并行处理。支持基于类药性性质的化合物库过滤和早期 ADMET 问题识别。

**典型使用场景**：
- 先导优化
- 化合物库过滤
- ADMET 预测
- 类药性评估

---

### 8. pytdc（治疗数据公共库）

**一句话简介**：提供治疗数据公共库（TDC）访问的 Python 库，包含药物发现和开发的精选数据集和基准。

**详细描述**：
PyTDC 包含 ADMET 预测（吸收、分布、代谢、排泄、毒性）、药物-靶点相互作用、药物-药物相互作用、药物反应预测、分子生成和逆合成的数据集。提供标准化数据格式、自动预处理的数据加载器、基准任务和评估指标、模型比较排行榜，以及与流行 ML 框架的集成。

**典型使用场景**：
- 药物发现 ML 模型的基准测试
- ADMET 预测模型开发
- 药物-靶点相互作用预测

---

### 9. rowan（云端量子化学平台）

**一句话简介**：具有 Python API 的云端量子化学平台，用于计算化学工作流。

**详细描述**：
Rowan 提供 45+ 化学计算：pKa 预测、氧化还原电位、溶解度、构象搜索、几何优化、蛋白质-配体对接（AutoDock Vina）和 AI 驱动的蛋白质共折叠（Chai-1、Boltz-1/2）。支持 DFT、半经验（GFN-xTB）和神经网络势方法（AIMNet2、Egret）。特性包括自动云资源分配、统一 API、RDKit 原生接口、工作流组织和批量处理。需要 labs.rowansci.com 的 API 密钥。

**典型使用场景**：
- 分子性质预测
- 基于结构的药物设计
- 虚拟筛选活动
- 蛋白质-配体结合预测
- 构象分析
