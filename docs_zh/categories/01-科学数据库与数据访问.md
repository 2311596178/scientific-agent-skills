# 分类一：科学数据库与数据访问

> 本分类包含 **6 个技能**，提供对 100+ 科学、生物医学、材料科学及经济数据库的统一或专用访问能力。

---

## 分类概述

科学研究离不开高质量的数据。本分类的技能覆盖了从化学分子数据库（PubChem、ChEMBL）、生物医学数据库（UniProt、ClinVar、COSMIC）到经济金融数据（FRED、美国财政部财政数据）的广泛领域。其中 **Database Lookup** 是一个统一的"万能查询"技能，可直接访问 78 个公共数据库的 REST API；而其他技能则针对特定专业数据库提供更深入的查询和分析能力。

---

## 技能清单

### 1. database-lookup（数据库统一查询）

**一句话简介**：通过 REST API 搜索 78 个公共科学、生物医学、材料科学及经济数据库，返回结构化 JSON 结果。

**详细描述**：
这是整个技能库中覆盖面最广的数据库查询技能。它整合了物理学/天文学（NASA、NIST、SDSS、SIMBAD、系外行星档案）、地球/环境科学（USGS、NOAA、EPA、OpenWeatherMap）、化学/药物（PubChem、ChEMBL、DrugBank、FDA、KEGG、DailyMed、ZINC、BindingDB）、材料科学（Materials Project、COD）、生物学/基因组学（Reactome、BRENDA、UniProt、STRING、Ensembl、NCBI Gene、GEO、GTEx、PDB、AlphaFold、InterPro、ChEBI、BioGRID、Gene Ontology、QuickGO、dbSNP、SRA、ENA、gnomAD、UCSC Genome、ENCODE、JASPAR、MouseMine、PRIDE、LINCS L1000、Human Protein Atlas、Human Cell Atlas、RummaGEO、Metabolomics Workbench、EMDB、Addgene）、疾病/临床（COSMIC、Open Targets、ClinPGx、ClinicalTrials.gov、OMIM、ClinVar、GDC/TCGA、cBioPortal、DisGeNET、GWAS Catalog、Monarch、HPO）、监管（FDA、USPTO、SEC EDGAR）、经济/金融（FRED、BEA、BLS、Federal Reserve、World Bank、ECB、US Treasury、Alpha Vantage、Data Commons）以及人口统计（US Census、Eurostat、WHO）等领域的数据库。

**典型使用场景**：
- 查询化合物的结构、性质和生物活性数据
- 检索基因、蛋白质序列和功能注释
- 查找通路、酶和基因表达数据
- 搜索临床试验、专利、SEC 文件
- 获取经济指标、晶体结构、天文对象、地震数据、天气信息

### 2. depmap（癌症依赖图谱）

**一句话简介**：查询癌症依赖图谱（DepMap），获取癌细胞系基因依赖评分、药物敏感性数据和基因效应谱。

**详细描述**：
DepMap 技能提供对 Cancer Dependency Map 数据库的查询能力。该数据库整合了 CRISPR Chronos 基因敲除筛选数据、药物敏感性实验结果以及基因效应评分。通过此技能，可以识别癌症特异性脆弱性、合成致死相互作用，并验证肿瘤药物靶点。数据覆盖数百种癌细胞系和数千个基因，是肿瘤学研究中靶点发现和验证的关键资源。

**典型使用场景**：
- 识别特定癌症类型的关键依赖基因
- 发现合成致死基因对（如 BRCA1/2 与 PARP）
- 验证候选药物靶点在癌细胞系中的重要性
- 分析药物敏感性与基因型之间的关联

---

### 3. imaging-data-commons（NCI 影像数据共享）

**一句话简介**：使用 idc-index 查询和下载 NCI Imaging Data Commons 的公共癌症影像数据。

**详细描述**：
NCI Imaging Data Commons (IDC) 是美国国家癌症研究所提供的公开癌症影像数据平台。该技能支持无需认证即可查询大规模的放射学（CT、MR、PET）和病理学数据集。用户可以通过元数据进行筛选、在浏览器中可视化影像、检查数据许可协议，并下载用于 AI 训练或研究的数据。

**典型使用场景**：
- 获取用于深度学习模型训练的公共癌症影像数据集
- 查询特定癌症类型（如肺癌、乳腺癌）的影像学数据
- 进行放射组学（radiomics）研究
- 开发和验证医学影像 AI 算法

---

### 4. primekg（精准医学知识图谱）

**一句话简介**：查询精准医学知识图谱（PrimeKG），获取基因、药物、疾病、表型等多尺度生物医学数据。

**详细描述**：
PrimeKG 整合了 20 多个生物医学资源，构建了一个统一的知识图谱。它连接了基因、药物、疾病、表型、通路、解剖结构等多类型节点，形成超过 400 万条关系边。该技能支持药物重定位、疾病机制探索和靶点识别等任务，是多组学数据整合和系统生物学研究的重要工具。

**典型使用场景**：
- 药物重定位（发现现有药物的新适应症）
- 疾病机制探索和靶点优先级排序
- 构建疾病-基因-药物关联网络
- 整合多源生物医学知识进行系统分析

---

### 5. usfiscaldata（美国财政部财政数据）

**一句话简介**：通过免费开放的 REST API 访问美国财政部的 54 个数据集和 182 个数据表。

**详细描述**：
美国财政部财政数据 API 提供联邦财政数据的全面访问，无需 API 密钥。涵盖的数据包括：国债数据（自 1993 年以来的每日国债数据、自 1790 年以来的历史国债）、每日财政部报表（TGA 余额、存取款）、月度财政部报表（联邦预算收入和支出）、国债拍卖数据（1979 年以来的国库券、国库票据、国债、TIPS、FRN）、国债平均利率、财政部报告汇率（170+ 货币季度数据）、I Bond 和储蓄债券利率、TIPS/CPI 数据等。支持过滤、排序、分页和 CSV/XML/JSON 输出格式。

**典型使用场景**：
- 宏观经济分析和财政研究
- 金融时间序列分析（国债收益率曲线）
- 汇率趋势分析和预测
- 政府支出和收入趋势研究

### 6. hugging-science（Hugging Face 科学资源目录）

**一句话简介**：Hugging Face 上科学数据集、模型、博客文章和交互式 Spaces 的精选 LLM 友好型目录。

**详细描述**：
Hugging Science 覆盖 17 个科学领域（天文学、基准测试、生物学、生物技术、化学、气候、保护、地球科学、生态学、能源、工程、基因组学、材料科学、数学、医学、物理学、科学推理）。发现过程通过 huggingscience.co 进行（提供 `llms.txt`、`llms-full.txt` 和按主题分类的 markdown 文件，专为智能体消费设计）；使用则通过标准 Hugging Face API 进行。包含 `fetch_catalog.py` 脚本，支持按主题、类型（数据集/模型/博客/Spaces）或自由文本搜索进行过滤访问。还包括加载数据集（支持流式处理十亿 token 语料库）、通过 `transformers` 或推理 API 运行模型（处理 Evo-2 等自定义架构的 `trust_remote_code` 要求）、通过 `gradio_client` 调用 Spaces（含 BoltzGen 蛋白质结合物设计示例）的参考指南。通过 `.env` 中的 `HF_TOKEN` 认证受限资源。

**典型使用场景**：
- 发现特定科学 ML 任务的合适数据集或模型
- 在精选科学数据上进行微调
- 引用数据集/模型作者的方法论博客
- 运行交互式科学演示（结合物设计、定理证明、天气建模）
- 从 "我需要蛋白质/基因组/分子/气候/材料/天文学的模型" 到可运行代码的桥接
